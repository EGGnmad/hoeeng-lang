# 호에엥 언어

> 고인물 HELP!!!

### 연산자

* `.` +1
* `~` -1
* `ㅇ` +
* `엣` *
* `엫` /

### 변수 대입
```python
흐..에..엣.....
#2번째 변수에 2*5 대입
```

### 출력
```python
호..에...
#(2번째 변수(10) + 3) = 13 출력 
```

```python
# 아스키 코드로 출력

호..에엣...ㅇ...엥
#(2번째 변수(10) * 3 + 3) = "!"(33) 출력 
```

### 입력
```python
흐..에헤
#2번째 변수에 입력한 숫자 대입
```
